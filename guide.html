Hướng Dẫn Vận Hành App Chính

🎬 Advance Audio-Video AutoSync

App cốt lõi — biến bộ nguyên liệu (Audio + Ảnh/Video + Kịch bản) thành video hoàn chỉnh chỉ với 1 nút bấm duy nhất.

Lưu ý Quan trọng

Các thông số kỹ thuật mặc định trên app đã được tối ưu hóa sẵn — không cần chỉnh thay khi mới dùng. Chỉ nạp đủ nguyên liệu và nhấn chạy.

CÁCH SỬ DỤNG — MỘT MÀN HÌNH, MỘT NÚT BẤM

  • Bước 1 — Chọn File Audio: Upload file âm thanh lồng tiếng đã chuẩn bị sẵn (xuất ra từ Tool TTS hoặc tự thu âm).
  • Bước 2 — Chọn File Kịch bản (.txt): File kịch bản có mỗi câu / mỗi cảnh nằm trên 1 dòng riêng. Số dòng phải khớp chính xác số ảnh/clip.
  • Bước 3 — Chọn Thư mục Ảnh / Video (đầu vào): Thư mục chứa các cảnh nền. Mỗi ảnh/clip tương ứng với 1 dòng kịch bản.
  • Bước 4 — Bấm nút: BẮT ĐẦU ĐỒNG BỘ
  • Bước 5 — Sau khi Quá trình đồng bộ kết thúc: Màn hình phân cảnh hiện ra, bạn có thể chọn ảnh/video nào sẽ hiển thị ở cảnh nào.
  • Bước 6 — Chọn Thư mục Đầu ra: Nơi lưu video thành phẩm sau khi app xử lý xong.
  • Bước 7 — Nhấn "Xuất Video": App sẽ tính toán thời lượng từng cảnh, điều chỉnh tốc độ (Speed/Trim/Loop) và xuất video hoàn chỉnh khớp âm thanh chính xác từng ms.

⚡ Tip: Import Nhanh Toàn Bộ 1 Lần

Nếu bạn đặt tất cả nguyên liệu (audio, kịch bản, ảnh/clip) vào cùng 1 thư mục, hãy nhấn Browse → "Thư mục tất cả nguyên liệu" ở góc phải trên — app sẽ tự động điền đầy đủ các ô và tự chọn Output. Không cần chọn từng thứ thủ công!

App Tặng Kèm — Visual Studio Pro (Whisk + 11Labs Pipeline)

Bộ công cụ hỗ trợ tạo nguyên liệu tự động từ kịch bản: tạo ảnh/video Whisk, lồng tiếng 11Labs. Tặng miễn phí kèm theo gói mua.

1. TAB STORYBOARD — PHÂN TÁCH KỊCH BẢN & TẠO PROMPT ẢNH

🔑 Chuẩn Bị — Lấy API Key Gemini

  1. Truy cập Google AI Studio.
  2. Đăng nhập Google → nhấn "Create API key" → sao chép toàn bộ mã.
  3. Mở Visual Studio Pro → Tab 1 (Storyboard) → dán vào khung API → nhấn "Kiểm tra & Lưu".
  • Nhập Kịch bản: Upload file txt hoặc paste toàn bộ kịch bản vào ô văn bản lớn để phân tách — mỗi phần tương ứng với 1 ảnh/clip trình diễn.
  • Tuỳ chỉnh phong cách: Lựa chọn Chủ đề (Anime, Cinematic...), Style Prompts, Tỷ lệ khung hình và Ngôn ngữ Đầu ra cho bộ ảnh.
  • Tạo Phân cảnh: Bấm "Bắt đầu phân tích & Chia dòng" để AI tự động cắt lớp kịch bản thành các Prompt sinh ảnh chi tiết.
  • Sau bước này các bạn nhớ tải file kịch bản .txt và file excel bằng cách bấm vào các nút ở góc trên bên phải của app nhé.

2. TAB WHISK — TẠO ẢNH & VIDEO

🔑 Chuẩn Bị — Lấy Cookie Whisk

  1. Cài tiện ích Cookie Exporter tại đây trên Chrome/Edge/Cốc Cốc.
  2. Truy cập labs.google/fx/tools/whisk — đăng nhập Gmail.
  3. Mở Cookie Exporter → nhấn "Copy as JSON".
  4. Mở Visual Studio Pro → Tab 2 (Whisk) → dán vào ô "Cookie (JSON)" → nhấn "Kiểm tra & Lưu".
  • Cài đặt sinh ảnh: Chọn loại mẫu sinh ảnh (Model), Tỷ lệ khung hình và thư mục lưu kết quả.
  • Thực thi tự động: Chọn số lượng ảnh muốn tạo cho MỖI prompt → nhấn "CHẠY NGAY". Quá trình sinh ảnh/video kết hợp tải về diễn ra hoàn toàn tự động, đa luồng. Bạn chỉ cần pha cafe và chờ đợi!

3. TAB 11LABS — LỒNG TIẾNG

🛡️ Chuẩn Bị — Cài ExpressVPN (bắt buộc)

ElevenLabs thường xuyên chặn IP lạm dụng API free của họ, bạn cần bật VPN trước khi sử dụng Tab này.

  1. Tải ExpressVPN: Nhấn vào đây để tải (Windows 64-bit).
  2. Cài đặt xong, mở ExpressVPN → nhấn "Sign In" → chọn "Use Activation Code".
    📖 Xem hướng dẫn chi tiết từng bước tại đây
  3. Nhập key kích hoạt: EWTD3CGRV3PRBT8O5IBBQYM
  4. Kết nối VPN → chọn bất kỳ server nào (Singapore, Mỹ, Nhật...) → Connect.

🔑 Chuẩn Bị — API Key ElevenLabs

Bộ công cụ đã tích hợp sẵn Pool 41 Key — nhấn nút bên dưới để copy toàn bộ, sau đó dán vào mục "Quản lý API Keys" trong app (mỗi dòng 1 key), rồi bấm nút THÊM, sau đó bấm Kiểm tra và chờ cho tool tự động kiểm tra toàn bộ các key này. Nếu dùng key cá nhân từ ElevenLabs, dán thêm vào danh sách đó.

✅ Đã copy! Dán vào app ngay.
  • Kịch bản Âm thanh: Paste kịch bản lời thoại vào ô "Kịch bản cần đọc".
  • Trợ lý Âm thanh: Khuyến nghị mạnh mẽ sử dụng Model v3 — thế hệ mới nhất của ElevenLabs, ngữ điệu cảm xúc tự nhiên như người thật, đọc tiếng Việt mượt mà.
  • Kiểm soát Ngắt Nghỉ: Tuỳ chỉnh thời gian nghỉ giữa các câu, đoạn văn. Chọn Auto Silence để AI tự động ngắt nghỉ tự nhiên nhất.
  • Xuất phẩm: Nhấn "BẮT ĐẦU TẠO AUDIO" — kết quả tự động lưu thành file mp3. Nghe thử ngay bằng Trình phát nhạc (Audio Player) tích hợp sẵn trên app.
